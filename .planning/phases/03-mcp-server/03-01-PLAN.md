# 03-01: MCP Server Core and Tools

## Objective
Implement the MCP stdio server `mcp/index.js` utilizing the official `@modelcontextprotocol/sdk`. Register the `run_audit`, `get_findings`, and `compare_screenshots` tools as requested in Phase 3 Context.

  </acceptance_criteria>
