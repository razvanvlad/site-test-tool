# 02-01: Express API Backend

## Objective
Create the Express server (`server.js`) that serves the static dashboard files and provides REST API endpoints to read audits/findings and update finding status from the SQLite database.

## Tasks

```xml
<task>
  <action>Install express: `npm install express`</action>
  <read_first>
    - package.json
  </read_first>
  <acceptance_criteria>
    - package.json contains "express" in dependencies
  </acceptance_criteria>
</task>

<task>
  <action>Create `server.js` with Express. Setup static file serving from `public/` directory (create the directory if missing). Setup JSON body parsing `app.use(express.json())`.</action>
  <read_first>
    - src/db.js
  </read_first>
  <acceptance_criteria>
    - server.js contains `express.static('public')`
    - server.js contains `app.listen(3000)`
  </acceptance_criteria>
</task>

<task>
  <action>Add GET `/api/audits` endpoint in `server.js` that returns all audits from the database ordered by `started_at` DESC.</action>
  <read_first>
    - server.js
    - src/db.js
  </read_first>
  <acceptance_criteria>
    - server.js contains `app.get('/api/audits'`
    - SELECT * FROM audits ORDER BY started_at DESC
  </acceptance_criteria>
</task>

<task>
  <action>Add GET `/api/findings/:auditId` endpoint in `server.js` that returns all findings for a specific audit ID.</action>
  <read_first>
    - server.js
  </read_first>
  <acceptance_criteria>
    - server.js contains `app.get('/api/findings/:auditId'`
    - SELECT * FROM findings WHERE audit_id = ?
  </acceptance_criteria>
</task>

<task>
  <action>Add PATCH `/api/findings/:id` endpoint in `server.js` that updates a finding's `status`, `is_false_positive`, and `notes`. Accept JSON body. Update the row in the `findings` table.</action>
  <read_first>
    - server.js
  </read_first>
  <acceptance_criteria>
    - server.js contains `app.patch('/api/findings/:id'`
    - UPDATE findings SET status = ?, is_false_positive = ?, notes = ? WHERE id = ?
  </acceptance_criteria>
</task>
```

## Verification
- Run `node server.js` and verify it listens on port 3000.
- Curl `http://localhost:3000/api/audits` and receive JSON.
