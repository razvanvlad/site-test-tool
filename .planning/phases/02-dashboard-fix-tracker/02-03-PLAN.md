# 02-03: Compare Script (Diff Capture)

## Objective
Implement `compare.js`, a script that runs a post-fix audit for a specific finding. It loads the finding's URL in Playwright, captures a new full-page screenshot (since Phase 1 before-screenshots are full-page), runs `pixelmatch` to generate a diff, and stores the results in the `fix_tracker` database table.

