# Plan 2: Audit Engines

<objective>
Implement the adapter wrappers for the 4 audit engines: Playwright (Console/Network/Screenshot), Lighthouse, axe-core, and Linkinator.
</objective>

<task>
<read_first>
- .planning/phases/01-cli-engine/01-CONTEXT.md
- site-test-tool-blueprint-v2.md
</read_first>
<action>
Create `src/engines/playwright-capture.js` to handle browser interactions.
1. Export a function `async captureSignals(url, page, options = {})`.
2. Attach listeners for `console`, `pageerror`, `requestfailed`, and `response`.
3. Capture errors into an array `consoleErrors` and failed requests into `failedRequests` (status >= 400).
4. Navigate to the URL using `page.goto(url, { waitUntil: 'networkidle', timeout: 30000 })` wrapped in a try/catch to avoid hard crashes on timeout.
5. Take a screenshot using `page.screenshot({ path: options.screenshotPath || 'reports/screenshots/temp.png', fullPage: true })`.
6. Return `{ consoleErrors, failedRequests }`.
</action>
<acceptance_criteria>
- `src/engines/playwright-capture.js` exports `captureSignals`.
- Contains `page.on('console'` and `page.on('pageerror'`.
- Contains `page.on('requestfailed'` and `page.on('response'`.
- Takes a screenshot using `fullPage: true`.
</acceptance_criteria>
</task>

<action>
Create `src/engines/lighthouse-runner.js` to handle Lighthouse.
1. Export a function `async runLighthouse(url)`.
2. Import `chromeLauncher` and `lighthouse`.
3. Launch chrome: `await chromeLauncher.launch({chromeFlags: ['--headless']})`.
4. Run lighthouse: `await lighthouse(url, {logLevel: 'error', output: 'json', onlyCategories: ['performance', 'accessibility', 'best-practices', 'seo'], port: chrome.port})`.
5. Kill chrome: `await chrome.kill()`.
6. Return `{ lhr: runnerResult.lhr }`. (Handle errors gracefully returning `{ error: ... }`).
</action>
<acceptance_criteria>
- `src/engines/lighthouse-runner.js` exports `runLighthouse`.
- Uses `chromeLauncher.launch`.
- Only runs categories: `performance`, `accessibility`, `best-practices`, `seo`.
</acceptance_criteria>
</task>
